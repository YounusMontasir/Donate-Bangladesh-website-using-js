What is the Document Object Model (DOM)?

DOM is used to create an interactive website. Which is like a tree and the document is the root of the tree. If you assume with basic HTML structure then we see that the document is the parent of the HTML object and HTML is the parent of the head and body.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation means attaching event listeners to the parent node instead of many child nodes. Suppose we create a ToDo list. Where we have to add or delete many tasks. If we attached event listeners with a child it may create a problem while deleting. If we attached event listeners with their parent div then the event is easily worked.
